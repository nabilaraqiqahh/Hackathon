# Database Setup Instructions

This folder contains the database `.sql` files for the project.

## How to Import the Database (For Team Members)

1. Make sure you have **XAMPP** installed and running (start both **Apache** and **MySQL**).
2. Open your web browser and go to `http://localhost/phpmyadmin`.
3. Click on **New** in the left sidebar to create a new database.
4. Name the database (e.g., `hackathon_db` - check with the team for the exact name) and click **Create**.
5. Select your newly created database from the left sidebar.
6. Go to the **Import** tab at the top.
7. Click on **Choose File** and select the `.sql` file located in this folder.
8. Scroll down and click **Import** (or **Go** depending on your phpMyAdmin version).
9. You should see a success message. Your database is now ready!
